# SEO Command Center

## Executive Intent
A single unified executive surface that aggregates technical health, rankings, entity dominance, provider stability, and ROI into a cohesive operational command layer.

## Definitions
Technical Score: Composite crawl health metric.
Entity Score: Competitive semantic dominance metric.
Platform State: Derived status (Healthy, Degraded, Critical).
Share of Voice: % presence in SERP relative to competitors.

## Required Data Inputs
- Crawl engine
- Rank engine
- Entity engine
- Observability
- ROI engine

## Required API
GET /api/v1/dashboard/command-center?campaign_id=...

## Output Sections
- KPI cards
- Trend graphs
- Alert feed
- Provider health summary
