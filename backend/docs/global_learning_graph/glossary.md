# Glossary

## Global Learning Graph (GLG)
Cross-campaign knowledge graph storing reusable intelligence relationships.

## Cohort context
Metadata describing scope for evidence applicability (industry, geo, segment, time window, etc.).

## Outcome strength
Signed magnitude representing estimated impact size of a relationship.

## Support count
Number of observations contributing evidence to an edge.

## Confidence
Model-estimated probability that a relationship is valid for its cohort context.

## Strategy transfer
Applying learned strategy evidence from one campaign/cohort to another.

## Cold-start intelligence
Initial recommendation quality for campaigns with limited local history.

## Lineage
Traceability from graph relationships back to source events and models.

## Freshness decay
Confidence reduction for relationships that lack recent supporting evidence.
