# Glossary

Automation action:
Executable operation derived from approved recommendation.

Baseline window:
Pre execution period used as reference for outcome comparison.

Confidence score:
Estimated likelihood of positive recommendation impact.

Evaluation window:
Post execution period where effects are measured.

Feature:
Derived metric built from one or more raw signals.

Learning feedback:
Persisted update to pattern or policy derived from outcomes.

Outcome:
Measured change in KPI or SEO metrics after execution.

Pattern:
Repeatable relation between feature conditions and outcomes.

Policy bundle:
Versioned set of recommendation scoring and risk rules.

Recommendation:
Action proposal generated by policy engine.

Reward signal:
Scalar effectiveness metric used to tune policy.

SEO signal:
Raw or normalized point in time campaign measurement.

Temporal signal store:
Historical storage layer for canonical signals.

Version hash:
Deterministic fingerprint of schema and transformation version lineage.
