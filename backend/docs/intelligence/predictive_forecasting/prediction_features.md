# Prediction Features

## Feature groups
- Campaign state: rank trend, traffic baseline, technical health, content velocity.
- Strategy: action type, scope, historical effectiveness.
- Pattern: active signatures and strengths.
- Industry: segment priors from Industry Intelligence Network.
- Graph: support/confidence/recency from Global Learning Graph.
- Simulation history: prior forecast and calibration residuals.

## Requirements
- Point-in-time correctness
- Deterministic transforms
- Drift monitoring and bounded normalization
