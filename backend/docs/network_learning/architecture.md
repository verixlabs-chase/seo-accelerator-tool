# Network Intelligence Architecture

The Global SEO Intelligence Network extends campaign-local learning into a guarded cross-campaign learning layer.

Components:

- `seo_flight_recorder.py`
- `causal_outcome_analyzer.py`
- `industry_similarity_engine.py`
- `experiment_network_engine.py`
- `global_intelligence_network.py`

The network runs after policy learning and strategy evolution so it can consume real mutation outcomes and persisted strategy experiments.
