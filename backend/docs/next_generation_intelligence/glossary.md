# Glossary

## Strategy transfer
Applying proven strategies from similar campaigns to a target campaign.

## Causal rule
A validated conditional statement linking interventions to expected outcome changes.

## Cohort context
A scoped context such as industry, geo, size, or maturity used for relevance matching.

## Point-in-time correctness
Guarantee that features used for inference/training reflect only information available at that time.

## Confidence calibration
Adjustment process ensuring predicted probabilities align with observed outcomes.

## Circuit breaker
Automated control that halts risky automation when abnormal failure signals are detected.

## Idempotent processing
Handling repeated events without creating duplicate side effects.
