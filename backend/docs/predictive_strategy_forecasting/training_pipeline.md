# Training Pipeline

## Inputs
- Recommendation outcomes
- Digital twin simulation records
- Industry model snapshots
- Graph-derived evidence features

## Stages
1. Training set assembly
2. Validation and leakage checks
3. Model training
4. Cohort-level evaluation
5. Calibration fitting
6. Registry publication

## Cadence
- Daily incremental retraining
- Weekly full retraining
- Drift-triggered emergency retraining
